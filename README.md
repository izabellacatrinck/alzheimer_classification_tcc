# Classificação de Alzheimer (DenseNet vs MedSigLIP)
### Este repositório contém a implementação desenvolvida no Trabalho de Conclusão de Curso (TCC), que investiga a aplicação de técnicas de aprendizado profundo para a classificação binária da Doença de Alzheimer (AD) em imagens de ressonância magnética estrutural (MRI).

O trabalho compara duas abordagens distintas:

DenseNet-121 com transferência de aprendizado, representando o paradigma supervisionado tradicional baseado em Redes Neurais Convolucionais (CNNs);
MedSigLIP, um Vision-Language Model (VLM) especializado para aplicações médicas, avaliado nos cenários zero-shot e few-shot.

O objetivo é analisar o potencial de modelos visão-linguagem em tarefas de neuroimagem médica, especialmente em cenários com disponibilidade limitada de dados rotulados.
